com.tmall.wireless.alpha

类 Task

  • java.lang.Object
    • com.alibaba.android.alpha.Task

  • public abstract class Task
extends java.lang.Object
    这个类将一个个关联的Task，组织成PERT网路图的方式进行执行。可以通过Task.ProjectBuilder 将Task组装成完整的Project，该Project可以直接执行，也可以嵌套在另外一个Project 中作为其中一个Task执行。 Created by zhangshuliang.zsl on 15/8/18.

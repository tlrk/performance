com.tmall.wireless.alpha

类 AlphaManager

  • java.lang.Object
    • com.alibaba.android.alpha.AlphaManager

  • public class AlphaManager
extends java.lang.Object

    这个类的职责是将由构造完成的Project配置不同的模式，满足不同的进程有不同的初始化的需要。

    这个类提供了两种配置Project的方式，一种是用Task.ProjectBuilder， 自己用代码构建一个具体的Project；另外一种方式是在XML配置文件中配置，这里直接传入配置文件即可。

    Created by zhangshuliang.zsl on 15/8/18.

      • start

        public void start()
        开始启动流程，这里会根据当前的进程执行合适的启动流程。挑选的过程如下：
        1.检查当前的进程类型，如果是主进程转入第2步，否则转入第4步。
        2.如果当前是主进程，检查是否单独为主进程设置了启动流程，如果设置了，则执行此流程，否则转入下一步。
        3.检查是否设置了适用所有进程的启动流程，如果有则执行之，否则不执行任何流程，任务结束。
        4.如果当前是非主进程，检查是否有单独为当前进程设置启动流程，如果有，则执行之，否则转入下一步。
        5.检查是否设置了适合所有非主进程的启动流程，如果有，则执行之，否则转入下一步。
        6.检查是否设置了适合所有进程的启动流程，如果有，则执行值，否则不执行任何流程，任务结束。
